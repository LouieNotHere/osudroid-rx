# Changelog for osudroid!relax v1.11 update (May 13, 2024)
the new update is here! this update brings the replacement of the pp system, and changes to the cursor!

# Changes
- Say hello to DRP! This new system calculates the new amount for your old scores, *and this doesn't affect no mod pp calculation.*
- The precise mod matters! With this mod, the pp calculation will now make the accuracy pp value increase as the purpose of this mod is to change the hit window precisely, rather than having no change in it.
- Cursor change! The cursor trail spawn rate is set to 60, which now spawns a cursor trail asset every usual frame (for 60hz screens)
- PP display update! This change only affects the real-time ~~PP~~ DRP counter, and the ~~PP~~ DRP display in the score results screen!
